1.本拡張の目的

ニコ生の応援コメントは、放送が終わるとスクロールの彼方に流れてしまいがち。それを 自分のパソコンの中だけで 拾って育てていける形にまとめ直したのが、この拡張だよ。

本拡張は、ニコニコ生放送における応援コメント・参加者情報を、利用者自身の端末上で可視化・記録することを目的としたブラウザ拡張機能です。第三者への情報集約や広告配信を目的としていません。

2.取り扱う情報の種類

扱ってるものは、要は「今見ているニコ生の画面に出ている情報」 と、それをまとめ直した記録、あと 拡張の設定値 の 3 種類だよ。

本拡張は、次の情報を端末上で取り扱います。

2-1. 視聴中の放送に関する情報

  • 視聴ページの URL、放送番組 ID（lv...）、放送タイトル・放送者名・配信者ユーザー ID
  • 視聴者としてログイン中の場合のニコニコユーザー ID・ニックネーム（ニコニコページ上に表示されているものを読み取ります）

2-2. 放送内のコメント情報

  • コメント本文、投稿時刻、コメント番号
  • 投稿者のニコニコユーザー ID（数値 ID または匿名ハッシュ IDa:xxxx）、表示名、アイコン画像 URL
  • ギフト・スタンプ等の付帯イベント情報
たぬ姉メモ：コメント本文には視聴者同士のやりとりが含まれますが、本拡張はこれをあなたの端末内で表示・集計する目的に限って利用します。

2-3. 本拡張から送信した自身のコメント

  • 本拡張の送信 UI から投稿したコメントの本文と時刻（自分のコメントを「応援アイコン」として識別するための照合用）

2-4. 拡張の設定値

  • 記録 ON/OFF、ポップアップ着せ替え、音声入力デバイス、匿名 Identicon の有無、各種 UI の開閉状態、など
  • OpenRouter 連携の有効/無効フラグ・API キー（opt-in で明示的に設定した場合のみ）

3.保存場所と保存期間

記録は Chrome の chrome.storage.local に保存されるよ。これは あなたのパソコン専用の保管庫 で、他の端末には自動で同期されないタイプの領域なんだ。

本拡張が扱う情報の保存先は、利用者の端末上の Chrome 拡張機能ストレージ（chrome.storage.local）に限ります。ブラウザアカウント経由の自動同期（chrome.storage.sync）は使用していません。

保存期間は利用者が能動的に削除操作を行うまで（または拡張機能をアンインストールするまで）です。利用者は拡張ポップアップ内の「キャッシュクリア」「記録クリア」操作、または Chrome の拡張管理画面からのアンインストールにより、いつでも全データを削除できます。

たぬ姉メモ：chrome.storage.local は ブラウザのサンドボックス領域で、他のウェブサイトや他の拡張機能からは読み書きできません。

4.外部への送信について

この拡張の開発元（Kimito-Link Project）のサーバーには何も送ってません。そもそも受け口のサーバーを持ってないよ。外に出るのは ニコニコ公式 と、あなたが自分で有効化した場合だけ OpenRouter の 2 つだけ。

本拡張は、当プロジェクトが運用するサーバーを有しておらず、利用者の情報を当プロジェクト側に送信・蓄積することは一切ありません。

外部ドメインへの通信は、以下の目的に限って発生します。

4-1. ニコニコ生放送（*.nicovideo.jp, *.nimg.jp）

  • 視聴中ページのコメント取得・アイコン画像取得（利用者が通常ブラウザで視聴する際と同等のリクエスト経路）
  • 利用者自身のニコニコアカウントに紐づく認証情報は、ブラウザが保持する Cookie によりニコニコ側に直接送信されるものであり、当プロジェクトがこれを取得・保持することはありません

4-2. OpenRouter（openrouter.ai）

  • 利用者が AI 人物推定機能を有効化し、同意フラグを取得した場合に限り、コメント本文等が OpenRouter API に送信されます（詳細は §6）

4-3. 開発者モード（127.0.0.1:3456）

  • 開発者が自端末でデバッグする場合にのみ動作する経路であり、一般利用者には関係しません

5.ニコニコから取得する情報

拡張は あなたが今開いてるニコ生のページ を横から覗いて、コメントや放送情報を読み取ってるだけ。裏でこっそり別の放送を見に行ったりはしない よ。

本拡張は、manifest.json の host_permissions にて https://*.nicovideo.jp/* を宣言しており、以下のリソースにアクセスします。

  • 視聴ページの DOM 内容（放送情報、コメントパネル）
  • WebSocket / NDGR（ニコニコのコメント配信経路）から受信されるメッセージ
  • ニコニコユーザーアイコンの公開 CDN（secure-dcdn.cdn.nimg.jp）上の画像

これらはいずれも、利用者が自身のブラウザでニコニコ生放送を視聴する過程で既に受信している公開情報、もしくは利用者自身のブラウザセッションで正規に取得可能な情報に限られます。

6.OpenRouter（AI 連携）の扱い

AI でコメントを読んで雰囲気タグを付ける機能は、デフォルトでオフ。自分で API キーを入れて、同意ダイアログにチェックを入れた人だけ使えるよ。オフのままなら 1 回も外に送られません。

本拡張は、OpenRouter（openrouter.ai）を経由した AI による人物推定タグ生成機能を備えています。本機能は以下の条件すべてを利用者が明示的に満たした場合に限り動作します。

  1. 利用者が自身で取得した OpenRouter API キーを本拡張の設定画面に入力していること（nls_openrouter_api_key_v1）
  2. 機能有効化フラグ（nls_openrouter_enabled_v1）が明示的に true に設定されていること（既定値は未設定＝無効）
  3. コメント本文の送信に関する同意フラグ（nls_openrouter_data_consent_v1）が、ダイアログ上での明示同意により true に設定されていること

上記 3 条件のいずれかが満たされていない場合、OpenRouter への API 呼び出しは発生しません。

6-1. 送信される情報

  • コメント本文、投稿者のニックネーム（匿名名義を含む）、解析対象のテキスト断片
  • API キー（認証ヘッダとして）

6-2. 送信先

  • OpenRouter（https://openrouter.ai/）を経由し、利用者が選択した AI モデルのプロバイダに中継されます
  • OpenRouter および各プロバイダの個人情報取扱いについては、各社のプライバシーポリシーをご確認ください

6-3. API キー・課金

  • OpenRouter の API キー、ならびにそれに紐づく課金は、利用者と OpenRouter との直接契約に基づくものです
  • 当プロジェクトは OpenRouter の課金・請求に関与せず、API キーを当プロジェクト側で取得・保持することもありません
  • API キーは利用者端末の chrome.storage.local にのみ保存され、chrome.storage.sync には保存しません（キー漏洩リスクを抑えるため）

6-4. 停止方法

設定画面で機能有効化フラグを false にするか、API キーを削除することで、以後の OpenRouter への送信を停止できます。すでに送信済みのデータについては、OpenRouter 側のポリシーに従います。

7.Cookie・トラッキング

この公式サイトも、拡張本体も、広告のトラッキングとか行動履歴の集計とかは一切やってません。Google Analytics とかの計測タグも入れてないよ。

本拡張および本ウェブサイト（https://tsuioku-no-kirameki.com/）では、広告配信・行動追跡・閲覧者分析を目的とした Cookie、ローカルストレージ、その他の追跡技術を使用していません。第三者の計測タグ（Google Analytics、各種広告 SDK 等）は組み込まれていません。

8.第三者提供・販売

記録を 他社に売ったり共有したりはしない よ。そもそも記録は各自の PC の中にあって、私たちが見られる場所にはないんだ。

当プロジェクトは、本拡張を通じて利用者が取り扱う情報を、第三者に販売・提供・共有することはありません。法令に基づく開示請求等がある場合を除き、利用者の端末に保存されているデータにアクセスする手段そのものを保持していません。

9.利用者の権利（削除・書き出し・開示）

自分の記録は 自分で消せるし、自分で書き出せる。「キャッシュクリア」で派生データ、「記録クリア」でコメント本体、アンインストールで全部ごっそり消えるよ。
  • 削除：拡張ポップアップ内の「キャッシュクリア」または「記録クリア」操作、もしくは Chrome の拡張管理画面からのアンインストールにより、本拡張が保存した全データを削除できます。
  • 書き出し：記録済みの応援コメント・集計データは、拡張の書き出し機能から JSON / HTML 形式でローカルファイルとして取得できます。
  • 開示請求：保存データは利用者端末にのみ存在するため、利用者自身によるアクセスが可能です。なお、当プロジェクト側には利用者データの保管・提供に関する開示義務の対象となる情報は存在しません。

10.セキュリティ

本拡張は、Chrome 拡張機能のセキュリティモデル（Manifest V3、サンドボックス、host_permissions 宣言）に従って動作します。OpenRouter 連携に関わる通信は HTTPS を用い、API キーは端末ローカルストレージにのみ保存されます。

なお、利用者端末自体の管理（OS・ブラウザの更新、マルウェア対策等）については、利用者の責任となります。

11.未成年の利用

本拡張は、13 歳未満の児童の個人情報を意図的に収集することを目的としていません。本拡張は、ニコニコ生放送の利用規約および年齢制限に従って利用されることを前提としています。保護者の方は、お子様の本拡張の利用にあたって、ニコニコ公式の規約および本ポリシーの内容をご確認のうえ、必要に応じて利用の可否をご判断ください。
