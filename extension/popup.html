    ページ内パネルの幅（watch）

    コメントの取得や送信がうまくいかないときは、watchページを再読み込み（F5）してください。拡張機能のファイルを差し替えたあとは、Chrome の chrome://extensions を開き、nicolivelog の「更新」を押してから、必要なら watch ページも再読み込みしてください（拡張は自動では再読み込みされません）。

    たぬ姉

    やあ…たぬ姉だよ。下の入力欄をクリックしてフォーカスしてから話すと、OS の音声入力で文字が入ることが多いよ。

    Windows なら…Win キーと H を同時に押すと、音声入力のバーが出ることが多いよ（バージョンで違うかも）。

    Mac なら…「システム設定 → キーボード → 音声入力」をオンにしてから、よくあるのは Fn を2回、か Control を2回、だよ。実際のショートカットは設定画面で確認してね。

    拡張の「音声入力」ボタンが見えないレイアウトでも、この方法なら使えることがあるよ。細かい手順は下の「ガイド」を開いてね。

    音声メインなら「Enter でも送信」にチェックを…キーボードで打つ人は Ctrl+Enter（Mac は Cmd+Enter）がおすすめだよ。

    PCの音声入力でも入れます（ガイド）

    下の入力欄をクリックしてフォーカスしたうえで、OS の音声入力を使うと、ブラウザの「音声入力」より安定することがあります。

    • Windows：Win + H で音声入力を開き、この欄に向かって話す（バージョンにより名称・設定が異なります）。
    • Mac：システム設定のキーボードで音声入力をオンにし、この欄にフォーカスした状態で話す。ショートカットは多くの環境で Fn を2回、または Control を2回（設定の「音声入力」パネルで変更・確認できます）。
